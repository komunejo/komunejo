title: Python 3.8+ with PyYAML is an accepted dependency of generated spaces
status: accepted
date: 2026-07-12
addresses: [REQ-002]
---

The deterministic guarantee is worth the runtime: spaces require Python 3.8+ and PyYAML, stated plainly at creation. No degraded no-Python mode is designed — integrity by convention was the failure mode this plugin exists to leave behind.
