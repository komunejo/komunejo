title: The kernel installs exactly four entity types
status: accepted
date: 2026-07-12
addresses: [REQ-001, REQ-004]

Profile (the collaboration’s configuration; name pending, [[DEC-012]]), Skill (a registered capability), Section (a unit of composable report content), Composition (a document type as an ordered list of section refs). Nothing else is kernel: every domain entity arrives later as particularization ([[DEC-008]]).
